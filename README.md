## Multiple Toko

Ini cuma contoh web app dengan dynamic endpoints.

## Setup

* Node js v14.16.1 keatas
* Yarn (`npm i -g yarn`)
* MySQL

```bash
# Setup di backend dulu
$ cd backend
$ yarn
# Sambil nunggu, ubah dulu MySQL user-password di file ubah_ini.js
$ yarn start

# Setup di frontend
$ cd frontend
$ yarn
$ yarn start
```
